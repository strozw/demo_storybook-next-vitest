# Demo App for Next.js with Storybook

## Setup

```
pnpm install
```

## Tasks

### Develop

```
pnpm dev
```

### Build APP

```
pnpm build
```

### Develop by Strobyook

```
pnpm storybook
```

### Build Strobyook

```
pnpm build-storybook
```

### Lint

```
pnpm test
```

### Test

```
pnpm test
```
